# 🐶 TinDog — Tinder for Dogs  
A playful and responsive landing page inspired by Tinder, designed for dogs to meet, match, and find love. Built with modern Bootstrap components, clean layout structure, and a fun brand identity.

---

## 📌 Overview  
TinDog is a single‑page responsive website that showcases a fictional mobile app where dogs can discover new friends, matches, and potential partners. The landing page highlights the app’s features, pricing plans, testimonials, and download options, all wrapped in a colorful and engaging design.

---

## 🎨 Features  
- Responsive hero section with app mockup and download buttons  
- Highlighted features with icons and short descriptions  
- Testimonial section with user quote and brand logos  
- Pricing plans with tiered options for different dog “profiles”  
- Footer with useful links, contact information, and social media  
- Gradient backgrounds and modern Bootstrap styling  

---

## 🧩 Sections  
- **Title / Hero** — App introduction, call‑to‑action, and device mockup  
- **Features** — Key selling points of the TinDog app  
- **Testimonials** — Social proof with a fun dog quote and brand mentions  
- **Pricing** — Three subscription tiers: Chihuahua, Labrador, Mastiff  
- **Footer** — Links, contact details, and social media  

---

## 🛠️ Technologies Used  
- Bootstrap 5  
- HTML5  
- CSS3  
- SVG icons  
- Responsive grid system
